06.java開發環境搭建
  1.java虛擬機(JVM):運行所有java程序的假想計算機,編寫的java代碼都運行在JVM(類似翻譯機)之上,實現java
    語言跨平台特性的重要特性(核心)
  2.JRE:是java程序的運行環境,包含JVM和運行時所需要的核心類庫(運行必需)
  3.JDK:是java程序開發工具包,包含JRE和開發人員使用的工具(範圍最大,開發必需)
  要運行一個已有的java程序,只需安裝JRE即可
  要開發一個全新的java程序,則需安裝JDK

  3.運行:java.exe(解釋程式) -> java 新生成的檔案(去除.class)  //Hello World!

   運行 java HelloWorld  -> 輸出運行結果

15.關鍵字的特點
  1.完全小寫的字母 如public、class、static
  2.有特殊顏色顯示

16.標識符(識別符):自定義名稱  如類名、變量名、函數名、檔案名
  1.可包含英文、數字、$及下底線_
  2.不可以數字開頭
  3.不可使用關鍵字
  4.類名使用大駝峰命名法:首字母大寫 HelloWorld
  5.變量名及方法名使用小駝峰命名法:首單詞字母小寫,其他單詞首字母大寫 helloWorld

16.常量(常數):在程序運行過程中,固定不變的值
  1. 字符串常量:用雙引號括起來的內容 "Hello World" "黑馬程序員"  ""
  2. 字符常量:用單引號括起來的單個字符  'A' '0' '中' 不可以為空或兩個以上字符 '' 或 'ab'
  3. 整數常量:不帶小數的數字 0 100 -120
  4. 符點數常量:帶小數的數字  0.0
  5. 布林常量:布林值,true/false
  6. 空常量:代表没有任何數據,空值 null **不能直接輸出**

18.數據類型
  1.1B(byte)=8bit
    1KB = 1024 B
    1MB = 1024 KB
    1GB = 1024 MB
    1TB = 1024 GB
  2.JAVA語言是強類型語言,對於每一種數據都給出了明確的數據類型,不同的數據類型也分配不同的內存空間,所以它們表示的數據大小也不一樣。
  3.分類:
    (1)基本數據類型:四類八種
      a.整數型:byte、short、int(默認)、long
      b.浮點型:float、double(默認)
      c.字符型:char
      d.布林型:boolean
    (2)引用數據類型:字符串、類(class)、接口(interface)、數組(array)
  4.注意事項:
    (1)字符串不是基本數據類型,而是引用類型
    (2)浮點型可能只是一個近似值,並非精確的值
    (3)數據範圍與字節數不一定相關,如float數據範圍比long更廣泛,但float是4字節,long是8字節
    (4)浮動數中默認類型為double,如果一定要使用float類型,需加上F後綴
       如果是整數,默認為int,如果一定要使用long類型,需加上L後綴。

19.變量:在程序運行過程中,其值可以發生改變的量,本質上,變量是內存中的一小塊區域
  1.定義變量:  格式: 數據類型 變量名 = 數據值   "=" 賦值,將右邊的值賦予給左邊的變量 注意:右邊的值不可以超過左側數據類型的取值範圍
  2.變量的使用:
    (1)取值  變量名
    (2)修改值  變量名 = 新值

21.變量使用注意事項:
  1.創建多個變量,變量名稱不可重複
  2.定義long和float類型變量時,記得加上後綴L和F。
  3.如果使用byte或short類型的變量,右側的數據值不能超過左側類型的範圍。
  4.没有賦值的變量,不能直接使用。
  5.變量使用不能超過作用域的範圍。
  作用域:從定義變量開始,一直到直接所屬的大括號結束為止。
  6.可以通過一個語句來創建多個變量,但一般情況下不推薦使用
  **同時創建三個變量並賦值
    int a=10,b=20,c=30;

22.數據類型轉換:等號左右兩側的數據類型不同時,區分自動轉換和強制轉換
  1.自動轉換(隠式):
    (1)代碼不需特殊處理,自動完成。
    (2)規則要求:
       數據範圍由小到大,等號右邊的值的範圍較小賦值給左邊範圍較大 byte -> short -> int -> long -> float -> double
    
  2.強制轉換(顯式):代碼需要特殊的格式處理,不能自動完成
    格式: 範圍小的類型  範圍小的變量名 = (範圍小的類型)大的類型 
         int  a = (int)100L  
    注意: 
      1.強制類型轉換不推薦使用,可能發生數據溢出(灑出來超出範圍,只能顯示該類型的最大數值)(整數而言):
        int num1=(int)600000000L  //結果是 1705032704 非600000000
        精度損失問題: double -> int 
        double num2=(int)3.14  // 3  非四捨五入,而是所有小數均捨去
      2.byte/short/chr三種類型可作數學運算,如加法
      3.byte/short/int三種類型在運算時,都會先被提昇為int類型,再參與運算
