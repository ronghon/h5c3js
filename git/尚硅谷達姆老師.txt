02.git簡史及安裝
    1.林納斯-托納茲(Linus)--linux及git作者--2005年
    2.支持免費開源
    3.分布式管理版本控制系統
    4.設計目標:2005年誕生
      (1)分支切換速度快(允許成千上萬並行開發的分支)
      (2)容量小,設計簡單
      (3)高效可管理大規模項目
    5.git --version 查看git版本
    6.環境配置-->查看配置現況使用git config --list
        git config --global user.name 'name'
        git config --global user.email 'email'
        system:系統,對系統中所有用戶都適用的配置,寫入檔案/etc/gitconfig
        global:全局,用戶目錄下只適用該用戶,寫入檔案~/.gitconfig
        local:只適用當前項目的git目錄中,寫入檔案.git/config
